Xameleon Web Server




Вы находитесь на сайте компонета для разработки легковесных REST сервисов.

  • Копмонент реализован на языке С++
  • Эта страница отдана браузеру с помощью Xameleon-Web-Server
  • Этот сервер является демонстрационной программой и одновременно тестами для библиотеки
  • Cтатус библиотеки: альфа-версия

Исходный код проекта находится на github.com.
